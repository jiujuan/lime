# Lime 文档站

Lime 是给中文创作者用的桌面端 AI 内容工作台。它把资料、灵感、生成、修改和复盘放在一个地方，让一次创作不是一次性聊天，而是一条能继续推进的工作流。

## 当前口径

当前面向创作者的对外文档默认采用以下口径：

1. 先讲创作者要完成什么：写文章、做脚本、整理资料、改版本、复盘结果
2. 先讲使用体验：资料和参考能沉淀，结果能继续改，常用做法能复用
3. 技术词只放在进阶页面，不放在普通用户的第一屏
4. 模型、连接器和开放接口都作为扩展能力说明，不能盖过创作主线

## 后续说明

后续恢复普通创作者向的入门文档时，应以仓库根目录 `README.md` 的表达为基线：先写创作者故事、使用流程和资料边界，再写进阶连接能力；不要回收旧版围绕技术结构展开的入门页面。
