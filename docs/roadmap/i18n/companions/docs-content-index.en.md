# Lime Docs

Lime is a desktop AI content workspace for Chinese creators. It keeps references, ideas, generation, revision, and review in one place, so a creative session can become a workflow that continues instead of a one-off chat.

## Current Positioning

The public docs currently use this positioning for creators:

1. Start with what creators need to finish: articles, scripts, research notes, revisions, and review
2. Start with the experience: references can be saved, outputs can keep evolving, and common methods can be reused
3. Keep technical terms in advanced pages instead of the first screen for general users
4. Treat models, connectors, and open APIs as extension capabilities, not as the main story
